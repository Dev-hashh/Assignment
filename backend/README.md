# Backend API

REST API for the assignment. It includes JWT authentication, role-based access control, tasks CRUD, validation, OpenAPI docs, and a Postgres schema.

## Run

```bash
npm start
```

## Test

```bash
npm test
```

## URLs

- API root: http://localhost:3000
- Health: http://localhost:3000/api/v1/health
- Docs: http://localhost:3000/docs
- OpenAPI: http://localhost:3000/openapi.json

Default admin:

- Email: `admin@example.com`
- Password: `Admin@12345`
